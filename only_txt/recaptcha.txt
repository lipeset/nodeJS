NODE É UMA PLATAFORMA BASEADA NO V8 - MOTOR DE EXECUÇÃO JAVASCRIPT DO CHROME
E ELE PERMITE RODAR JAVASCRIPT DO LADO DO SERVIDOR.

EXPRESS É UM FRAMEWORK PARA DESENVOLVIMENTO DE SISTEMAS WEB PARA O NODEJS.

EJS = EMBEDDED JAVASCRIPT TEMPLATES - PERMITE A CRIAÇÃO DE ARQUIVOS QUE CONTENHAM
HTML FIXO JUNTAMENTE COM CÓDIGOS JAVASCRIPT. OU SEJA, É UMA ENGINE QUE PERMITE
O PROCESSAMENTO DE PÁGINAS E QUE PERMITE QUE O JAVASCRIPT SEJA UTILIZADO DE UMA 
FORMA BEM SIMPLES DENTRO DESSES ARQUIVOS.

NODEMON É UM UTILITÁRIO QUE FICA CONSTANTEMENTE VERIFICANDO SE TEVE ALTERAÇÃO
EM ALGUM DE NOSSOS SCRIPTS E CASO EXISTA, ELE RAPIDAMENTE REINICIA O SERVIDOR.
UM UTILITÁRIO MAIS VOLTADO PARA O AMBIENTE DE DESENVOLVIMENTO. NÃO FAZ PARTE
DE FATO DA NOSSA APLICAÇÃO.

MYSQL - BANDO DE DADOS RELACIONAL. APRENDEMOS A FAZER A IMPLEMENTAÇÃO DO MÓDULO
BEM COMO A INSTALAÇÃO DO SERVIDOR LOCAL PARA REALIZARMOS NOSSOS TESTES.

TUDO ISSO FOI FEITO COM O NPM - NODE PACKAGE MANAGEMENT - O GERENCIADOR DE 
PACOTES DO NODEJS.

DENTRO DO DESIGN PATTERN MVC - MODEL; VIEW; CONTROL
SEMPRE MODULANDO A NOSSA APLICAÇÃO PARA O MAIS DESACOPLADO POSSÍVEL. ISOLANDO
ASSIM CADA PARTE EM DIRETÓRIOS SEPARADOS. TUDO O QUE FAZ PARTE DA APLICAÇÃO
ESTÁ NA PASTA 'app'; TUDO O QUE CONFIGURA O SERVIDOR ESTÁ NA PASTA 'config';
OS MÓDULOS INSTALADOS VIA NPM ESTÃO NA PASTA 'node_modules'.
